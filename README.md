### :pencil: Comandos
<pre>
    git clone https://github.com/juniorfdl/DesafioLaravel.git
    cd DesafioLaravel
    docker-compose up -d --build
    
    docker exec -it -u root app bash
    composer install
    php artisan key:generate
    php artisan migrate
    php artisan serve
</pre>
